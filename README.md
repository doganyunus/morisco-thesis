# Moriscos and the Ottoman Empire

A visual scrollytelling portfolio for the PhD dissertation *Moriscos and the Ottoman Empire: Entangled Histories in the Early Modern Mediterranean (1520–1620)* by Yunus Doğan.

**Live site:** https://doganyunus.github.io/morisco-thesis

**Stack:** Plain HTML · CSS · Vanilla JS · GSAP 3 (CDN) · No build step · No npm

**Pages:**
- `index.html` — primary scrollytelling narrative
- `researcher.html` — CV and academic profile
- `sources.html` — archives, methodology, glossary

**Contributing / agent rules:** See `agents.md` before making any changes.
